# GST Calculator Pro

A premium, fintech-grade application designed for blazing-fast Goods and Services Tax (GST) calculations and professional invoice generation. Built with a focus on exceptional user experience, high-performance architecture, and beautiful responsive design.

## 🚀 Features

- **Advanced GST Calculator**: Real-time extraction and addition of GST with intelligent exclusive/inclusive mode switching.
- **Dynamic Invoice Generator**: Create, preview, and download professional PDF invoices instantly.
- **Smart History Tracking**: LocalStorage-powered persistent history for both calculations and generated invoices.
- **Premium UI/UX**: 
  - Smooth 60fps Framer Motion animations.
  - Custom-branded orbital pre-loaders.
  - Fintech-style typography and dynamic number interpolations.
  - Seamless Dark/Light mode support with immediate theme hydration.
- **Progressive Web App (PWA)**: Fully installable with offline support for uninterrupted productivity.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4 + Framer Motion
- **PDF Generation**: jsPDF + html2canvas
- **Icons**: Lucide React
- **Architecture**: Component-driven with robust custom hooks and centralized core services.

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Subramanya-Veeregowda/GST-Calculator-Pro.git
   cd GST-Calculator-Pro
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design System

GST Calculator Pro utilizes a highly customized Tailwind configuration with semantic color tokens. It guarantees high contrast, accessibility, and a modern aesthetic across both mobile and desktop views.
